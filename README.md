# vue-typescript-boilerplate

This is vue typescript boilerplate.
With this boilerplate, you can easily start vue using class based syntax to make fully typed vue project.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
